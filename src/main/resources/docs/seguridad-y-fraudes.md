# Seguridad y Prevención de Fraudes

## Buenas prácticas para el cliente

- Nunca compartir:
    - Clave de cajero.
    - Claves de banca por internet o banca móvil.
    - Códigos de verificación enviados por SMS o correo.
- Verificar siempre:
    - La dirección web del banco (que sea la oficial).
    - El candado de seguridad en el navegador.
    - Que el cajero automático no tenga dispositivos extraños.

## Fraudes frecuentes

1. **Phishing**
    - Correos o mensajes falsos que imitan al banco.
    - Buscan que el cliente ingrese sus claves en una página falsa.

2. **Vishing**
    - Llamadas telefónicas donde alguien se hace pasar por el banco.
    - Piden claves o códigos “por seguridad”.

3. **Smishing**
    - Mensajes de texto con enlaces maliciosos.
    - Promesas de premios o alertas falsas sobre bloqueos.

## Responsabilidades y bloqueos

- Si el cliente sospecha de fraude, debe:
    - Bloquear sus tarjetas de inmediato.
    - Cambiar sus contraseñas.
    - Contactar al banco por los canales oficiales.

- El banco puede:
    - Bloquear preventivamente la tarjeta si detecta operaciones inusuales.
    - Solicitar información adicional para verificar la identidad del cliente.
